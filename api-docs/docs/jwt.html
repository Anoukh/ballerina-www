public object Header

Represents a JWT header

Field Name Data Type Default Value Description
alg string
typ string
cty string
kid string
customClaims map []

public object JWTIssuerConfig

Represents JWT issuer configurations

Field Name Data Type Default Value Description
certificateAlias string
keyPassword string

public object JWTValidatorConfig

Represents JWT validator configurations

Field Name Data Type Default Value Description
issuer string
audience string
certificateAlias string
timeSkew int

public object Payload

Represents a JWT payload

Field Name Data Type Default Value Description
iss string
sub string
aud string[] []
jti string
exp int
nbf int
iat int
customClaims map []

public function issue(Header header, Payload payload, JWTIssuerConfig config) returns (string | error)

Issue a JWT token

Parameter Name Data Type Description
header Header Header object
payload Payload Payload object
config JWTIssuerConfig JWTIssuerConfig object
Return Type Description
string | error string: JWT token string

public function sign(string data, string algorithm, string keyAlias, string keyPassword) returns (string)

Sign the given input jwt data.

Parameter Name Data Type Description
data string Original that need to sign.
algorithm string Signature string.
keyAlias string Private key alias. If this is null use default private key.
keyPassword string Private key password.
Return Type Description
string Signature. Signed string.

public function validate(string jwtToken, JWTValidatorConfig config) returns ((boolean,Payload) | error)

Validity given JWT token

Parameter Name Data Type Description
jwtToken string JWT token that need to validate
config JWTValidatorConfig JWTValidatorConfig object
Return Type Description
(boolean,Payload) | error boolean: If JWT token is valied true , else false

public function verifySignature(string data, string signature, string algorithm, string keyAlias) returns (boolean)

VerifySignature the signature of a given jwt.

Parameter Name Data Type Description
data string Original data which has signed.
signature string Signature string.
algorithm string Signature algorithm.
keyAlias string Public key alias.
Return Type Description
boolean Verified status. true or false.
