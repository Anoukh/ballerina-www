WHY WE BUILT BALLERINA >

The future apps we will write increasingly depend upon programmable endpoints. Integration is the discipline of resilient communication between endpoints. Integration is a domain of hard problems.

Ballerina is a simple programming language whose syntax and runtime addresses the hard problems of integration. As a Turing complete language, Ballerina enables agility with edit, build, run cycles. Ballerina code is compiled into services that include transactions, embedded brokers and gateway runtimes.
